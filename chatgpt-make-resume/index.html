利用ChatGPT助力程序员打造完美英文简历

前言

撰写英文简历一直是许多人的难题，尤其是对于非英语母语者来说，如何在确保语法正确的同时又让简历内容吸引人，确实是一项挑战。尽管网络上有很多教学文章，但看完之后仍然常常感到无从下手，不知该如何准确描述自己的经历和技能。因此，很多人最终选择寻求专业的简历编修服务。然而，现在有了ChatGPT这类先进的AI工具，我们再也不用为简历编修而烦恼了。

本文将详细讲解如何利用ChatGPT来辅助自己修订英文简历。我们将通过具体的例子，分享应该向ChatGPT提出哪些问题、下达什么指令，以便让它更有效地帮助你完善简历。今后，只需简单地对ChatGPT提出需求，它便能迅速给出建议，帮助你快速修改简历，同时还能节省不少编修费用。

为了实际测试ChatGPT在修改简历方面的效果，我们选取了一位前端工程师公开简历的前端重点部分作为示例，请ChatGPT协助进行修改。接下来，我们将以这个前端工程的例子为基础，进行深入探讨。

使用正确的指令

以下是几个简单的步骤，可帮助您轻松将中文简历转换成一份经过精心修订的英文简历：

  1. 首先，将中文简历的要点输入到Google翻译中，初步翻译成英文。值得注意的是，虽然ChatGPT也具备中英文翻译功能，但考虑到其处理中文的速度相对较慢，我们在此选择先使用Google翻译进行初步转换。

  2. 把翻译好的建立输入ChatGPT，然后在后面跟上Critique the following experience on a resume，这句的意思是，希望ChatGPT能够协助审查英文简历，并指出哪些地方可以进一步优化以提升表达效果。

  3. ChatGPT 回复后，接着输入 Rewrite the above resume bullet points using the suggestions you provided。这句话的含义是，希望ChatGPT能根据之前提供的建议，对简历中的内容进行改写，以优化表达并提升简历的整体质量。

  4. 完成简历的第一版改写后，我们或许还能发现一些可以进一步优化的地方。此时，我们可以再次请ChatGPT协助进行深度优化。一份优秀的简历，通常会用过去式强动词来开头描述经历，这能够凸显出个人的行动力与成果。同时，简历中也需要包含质性描述或量化数据来支持个人经历的描述，这有助于让招聘者更直观地了解我们的工作成果和能力。这时输入 Polish the above with strong action verbs 或 Polish the above quantitatively and qualitatively 请 ChatGPT 进一步用强动词改写，以及进一步在优化质性描述或量化数据的撰写。

  5. 有时候，ChatGPT在润色简历时可能会过于投入，导致生成的句子较长。为了保持简历的简洁明了，我们可以使用命令Make it more concise请求ChatGPT帮助我们缩短句子。

范例：前端工程

在了解上述步骤后，我们现在就来实际操作一番，看看如何运用这些技巧来优化我们的简历。以下是简历中能力的描述：

  • 使用 React、Tailwind CSS 作为基础进行开发线上编辑器
  • 基础网页功能与 RWD 切版设计
  • 设计 Server-side 渲染、设置 OG(Open Graph) meta tag 来提升网站 SEO
  • 以 Fabric.js 为本体实现编辑器拖曳功能
  • Redux 整合多个复杂 Reducer，并使用 immer.js 进行状态更新

先让Google帮我们翻译一下

把翻译后的内容输入到 ChatGPT，然后输入命令 Critique the following experience on a resume 请它帮忙给回馈。

然后输入 Rewrite the above resume bullet points using the suggestions you provided 请 ChatGPT 帮忙针对它自己提出的建议，来进行第一轮的修改。

第一轮的修改虽然对简历进行了初步的润色，但在质性描述与量化数据方面还有所欠缺，所以输入 Polish the above quantitatively and qualitatively 请 ChatGPT 进行第二轮修改。可以感受到 ChatGPT 厉害的地方，他会根据你提供的点，帮你想一些可以加的量化数据点。

举例来说，原本的简历上有提到优化 SEO，这时 ChatGPT 很准确地抓到 resulting in an increase in organic traffic by 20% (提升自然搜寻流量达 20%) 这个跟 SEO 相呼应的数据。当然这边提到的 20% 是 ChatGPT 捏造的，所以要记得把它改成实际的数据。

但是改完之后有点太长了，可以输入 Make it more concise 来请 ChatGPT 帮忙缩短内容。

等ChatGPT 缩完后，你会得到简短的版本。当然，您完全可以根据个人喜好和简历排版需求，自由组合长版本和短版本的描述，以形成最适合您简历的呈现方式。我这次使用两个版本综合出以下的版本：

  • Developed a feature-rich online editor using React and Tailwind CSS, boosting user engagement by 50%
  • Designed and executed the project with a focus on user experience, resulting in a clean and intuitive layout with responsive web design
  • Improved SEO with server-side rendering and OG meta tags, resulting in a 20% increase in organic traffic
  • Utilized Fabric.js to create a seamless drag and drop functionality for the editor, contributing to a user-friendly experience that reduces editing time by 40%.
  • Integrated multiple complex state management features into the application using Redux and immer.js, significantly improving code maintainability

与最初的中文版本相比，经过ChatGPT精心编修的英文版本，每句都以不同的强动词开头，同时辅以质性与量化的支持论点，使得原本的经历焕发出更为专业的光彩。

小结

通过上述实例可见，经过Google翻译与ChatGPT的润色后，简历的专业度得到了显著提升，仿佛经过专业简历编修服务的手笔一般。当前市场上，简历编修服务的价格不菲，便宜的也要上千，贵的更是高达三四千。然而，借助ChatGPT这类AI工具，我们不仅能够获得高质量的简历编修效果，还无需花费任何额外费用。

尽管ChatGPT能够协助我们优化简历的表述方式，但简历中的核心经历仍需我们自身不断积累。即使两份简历在写法上都达到了最佳状态，量化数据的差异也可能成为决定面试机会的关键因素。例如，一个处理过千万流量等级的经验，即便在表达方式上与只处理过十万流量的经历相似，也会因其显著的业绩而更容易获得面试机会。因此，我们在提升简历写作技巧的同时，更应注重自身经历的积累与提升，以便在求职过程中更具竞争力。
